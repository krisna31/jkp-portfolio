# Jelvin Krisna Putra Portfolio & Apps

## Setup

1. `git clone`
2. `cd jkp-portfolio`
3. `cp .env.example .env`
4. edit the .env following the local configuration
5. `composer install`
6. `npm i`
7. `npm run build`
8. `php artisan migrate:fresh --seed`
9. `php artisan shield:generate --all` (if any changes)
10. `php artisan shield:super-admin`

## Technology
1. Laravel
2. Filament
